# Resume

A clean, minimal, single-page resume built with plain HTML and CSS.

**Live site:** [sayyidsyamil.github.io/resume](https://sayyidsyamil.github.io/resume)

## Features

- Single HTML file — zero dependencies
- Responsive design (mobile & desktop)
- Print-friendly — press `Ctrl+P` / `Cmd+P` to save as PDF
- Easy to update — edit text directly in `index.html`
- SEO-optimized with structured data, meta tags, and semantic HTML

## How to Update

1. Open `index.html` in any text editor
2. Replace placeholder text with your info
3. Commit and push changes

## Hosting

This site is hosted for free on **GitHub Pages**.

To enable Pages:
1. Go to repo **Settings** → **Pages**
2. Source: **Deploy from a branch** → `main` → `/ (root)`
3. Save and wait ~1 minute
